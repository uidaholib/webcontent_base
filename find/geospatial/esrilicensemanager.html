Esri License Manager

A License Manager is available for ArcGIS for Desktop and/or Esri CityEngine.

  • If you are using a computer that is on the University of Idaho network:
    1. Follow the steps just below to connect to the license manager. Once connected you have the option of borrowing a license.
  • If you are using a computer that is not on the University of Idaho Network:
    1. Install the appropriate Cisco AnyConnect VPN client and then follow the steps just below to connect to the license manager     OR
    2. Contact Bruce Godfrey for a single-use license file

Connecting

1. If you are using a computer that is not on the UI network install the appropriate Cisco AnyConnect VPN client before proceeding.

Connecting to the License Manager Immediately Following the Installation of ArcGIS for Desktop

After ArcGIS for Desktop is installed you will be prompted to authorize the software.

1. Choose the radio button 'Advanced (ArcInfo) Concurrent Use'.

2. Push the 'Change...' button and enter 'lic-esri.its.uidaho.edu'.

3. Press 'Apply'.

 

 

Connecting to the License Manager Anytime after the Installation for ArcGIS for Desktop

Anytime after the installation of ARcGIS for Desktop you can start the 'ArcGIS Administrator' on your computer [All Programs > ArcGIS].

1. Select the 'Desktop' product. (Select 'CityEngine' for that product).

2. Choose the radio button 'Advanced (ArcInfo) Concurrent Use'.

3. Push the 'Change...' button and enter 'lic-esri.its.uidaho.edu'.

4. Press 'Apply' then 'OK'.

 

Borrowing

This is an optional step. License borrowing allows users to temporarily take concurrent use features from the License Manager on the license server and install them locally on their machines to use when off the University of Idaho network.  If the licenses are not returned within the borrowed time, they will automatically be disabled on the local machine and reenabled in the License Manager on the license server.

Start the 'ArcGIS Administrator' on your computer [All Programs > ArcGIS].

1. Select 'Borrow/Return' for either product.

2. Enter the number of days to borrow. The maximum is 365 days. Note that licenses expire every year in mid-February. Therefore, you will have to return and borrow a new license each year.

3. Check a box for each desired feature.

4. Press 'Apply' then 'OK'.

5. Optionally, at this point, you may wish to select the software product 'Advanced (ArcInfo) Single Use' in order to be able to view license expiration dates.

 

To return a license, uncheck the box associated with the desired feature(s) and press 'Apply' and 'OK'.
